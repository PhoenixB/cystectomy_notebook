4.2 Intraoperative blood loss

We estimated blood loss by surgical suction and visual assessment in milliliters. Blood loss ratio was calculated by absolute blood loss divided by estimated blood volume, as described previously7.

4.3 Statistical analysis

Continuous variables are presented as median [IQR] and categorical variables as frequencies. Group comparisons were done using the \(\chi^2\) test for categorical and the Kruskal-Wallis test for continuous variables. To identify independent risk factors for intraoperative blood product transfusion, we performed multiple logistic regression analyses. To avoid overfitting and assess model performance, we first partitioned data into training (90%) and hold-out sets (10%). Variable selection was done using a stepwise selection procedure minimizing AIC8 using the MASS package for R9. We evaluated the following variables as predictors: blood loss ratio, use of oral anticoagulants or platelet inhibitors, preoperative hemoglobin, BMI, age, sex, Charlson comorbidity index (CCI), tumor stage, positive nodal stage, duration of surgery, presence of previous surgery, use of noradrenaline, intraoperative crystalloids volume and presence of neoadjuvant chemotherapy and all two-way interactions between blood loss ratio, preoperative hemoglobin, age and gender. All continuous variables were normalized. Multicollinearity between predictors was checked using the variance inflation factor. Marginal means were calculated for ordinal categorical predictors using the emmeans package. Model fit was assessed with the le Cessie-van Houwelingen-Copas-Hosmer global goodness of fit test10, the Hosmer-Lemeshow test, binned residual plots (performance package) and GiViTI calibration plots (givitiR package)11.
